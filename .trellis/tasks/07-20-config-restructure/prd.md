# 配置体系重组 + 工作流优化

## Goal

重组 aqua 交互:欢迎页精简 + 工作主页顶部工具栏(替代 macOS 菜单栏与窗口分离、点菜单远的问题)+ 项目只读/加解锁(防误改)+ 配置中心/数据集独立页 + 驱动管理弹窗 + 去全局枚举 + 菜单精简。让配置组织清晰、触手可及,打开项目默认安全只读。

## 背景

- macOS 菜单栏与窗口分离,点菜单不顺手;配置 6 项分散(弹窗/页签混合),组织不清。
- 打开已有项目默认可编辑,易误改。
- 配置分层(已确认):
  - **应用级**:驱动管理(原"数据库管理"改名)+ 密钥,存 `app_data_dir`
  - **项目相关外置**:数据源,按项目路径单独存 + 加密
  - **项目级**(主 JSON 内):项目设置 / 业务类型 / 数据集;(全局枚举待删)

## 需求

### 1. 欢迎页(Welcome)
去掉"从数据库导入"按钮(没用,导入需先有项目),保留新建/打开两个按钮。

### 2. 工作主页工具栏(AppLayout 有项目时顶部常驻,非表编辑器)
- 操作组:`[导入]`(弹窗,只读时 disabled)`[导出]`(弹窗)
- `[加/解锁]` 切换项目只读
- 管理组:`[配置中心]`(独立页)`[数据集]`(独立页)`[驱动管理]`(弹窗)

### 3. 项目只读机制
- 打开已有项目默认**只读**(不可编辑);新建项目默认**可编辑**
- 工具栏加/解锁按钮切换
- 只读禁用编辑:字段/索引/表/分组的增删改、业务类型/数据集/项目设置编辑、导入
- 只读允许:导出、数据源/驱动管理(外置/应用级,与项目内容无关)

### 4. 配置中心(独立页 /config)
- 左侧返回按钮 -> 工作主页
- 内部左侧导航:项目设置 / 数据源 / 业务类型 + 右侧内容(迁移现有 ProjectSettingsDialog/DataSourceDialog/BizTypeManage 内容)

### 5. 数据集(独立页 /dataset)
- 左侧返回按钮 -> 工作主页
- 改造现有 DatasetManage(加左侧返回)

### 6. 驱动管理(弹窗)
- 工具栏 `[驱动管理]` -> 弹窗
- 原 DatabaseConfigDialog 改名"驱动管理"(实为 JDBC 驱动安装/显隐)

### 7. 全局枚举全删
- 删 EnumManage 页 + /enum 路由
- 删 `Project.enums` 数组(schema)
- 删 `FieldEnum::Ref` 引用机制,字段枚举统一内联 `InlineEnum`
- 同步:diff 的 enums diff、generator 枚举生成、validate、导入(enum_ref)、FieldDetailDialog 枚举编辑

### 8. 菜单精简
- 保留 macOS 系统菜单,精简为:文件(新建/打开/保存/另存/关闭/退出)+ 帮助(指南/关于)
- 导入/导出/配置/数据集/驱动 不再占菜单(移至工具栏)
- macOS 保留应用菜单(about/quit)

### 9. 关闭项目/关闭应用前检查 dirty
- 关闭项目:closeProject 已 confirmIfDirty(保留,验证有效)
- 关闭应用(窗口关闭/系统退出):listen 窗口 close 事件,dirty 时提示保存(confirmIfDirty),可取消关闭

## 验收标准

- [ ] 欢迎页仅新建/打开两按钮
- [ ] AppLayout 有项目时顶部工具栏,按钮直达:导入/导出(弹窗)、配置中心/数据集(独立页)、驱动管理(弹窗)、加/解锁
- [ ] 打开项目默认只读,新建可编辑,加/解锁切换;只读时字段/索引/表/分组/业务类型/数据集/项目设置/导入 禁用,导出/数据源/驱动管理 可用
- [ ] 配置中心 /config:左侧返回 + 左侧导航(项目设置/数据源/业务类型)+ 右侧内容,三项功能可用
- [ ] 数据集 /dataset:独立页 + 左侧返回,功能不回归
- [ ] 驱动管理:弹窗,标题"驱动管理",功能不回归
- [ ] 全局枚举:EnumManage 页 + /enum 路由删除;Project.enums 字段删除;FieldEnum 只剩 Inline;diff/generator/validate/导入/字段编辑同步;字段枚举内联可用
- [ ] 菜单精简为 文件 + 帮助(含 macOS 应用菜单)
- [ ] 关闭项目/关闭应用时,dirty 提示保存(保存/不保存/取消)
- [ ] 现有功能不回归:导入/导出/数据源 CRUD/业务类型 CRUD/数据集 CRUD/驱动管理/项目设置
- [ ] 布局遵守 box-sizing: border-box(reset),工具栏 flex-shrink-0,内容区 flex-1 min-h-0

## 非目标

- 数据源/业务类型/数据集的内部功能改造(仅迁移到新容器,不改逻辑)
- 驱动管理功能改造(仅改名+改入口)
- 新增配置项
